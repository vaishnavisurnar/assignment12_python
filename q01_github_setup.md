# Q1. GitHub Account Setup

1. Create a GitHub account.
2. Create a new repository named `Python-Basic-Projects`.
3. Add the assignment files to the repository.
4. Commit each completed feature with a clear message, for example:
   - `Create analyze_string function`
   - `Implement manage_marks with exception handling`
   - `Add smart calculator mini project`
5. Push the completed project to GitHub.

Useful commands:

```bash
git init
git add assignments/session_12_aiml_assignment
git commit -m "Add Session 12 AIML assignment solutions"
git branch -M main
git remote add origin YOUR_GITHUB_REPOSITORY_URL
git push -u origin main
```
